## 사용된 기술
* Retrofit2 라이브러리로 Restful API 구현
* Glide 라이브러리로 이미지 파일 데이터 로딩
* GMS Location 라이브러리로 위치확인기능 구현
* RecyclerView로 리스트 구현
* Spinner로 드롭다운 메뉴 구현
* CalendarView로 날짜 선택 구현<br/><br/><br/>
